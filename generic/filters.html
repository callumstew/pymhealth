Generic¶

Generic preprocessing functions that may apply to any or multiple signals

Filters¶

Butterworth¶

butterworth(arr, cutoff, freq, order=5, ftype='highpass')
Generic Butterworth function
Parameters

  • acc (float[n]) – Signal vector

  • cutoff (float or (float, float)) – Cutoff value(s)

  • freq (float) – Sampling frequency of signal

  • order (int) – Order of Butterworth filter. Default - 5

  • ftype (str) – Filter type {‘highpass’, ‘lowpass’, ‘bandpass’} Default - ‘highpass’

Return type

float[n]
