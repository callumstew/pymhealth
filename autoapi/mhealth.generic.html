mhealth.generic.filters module¶

Generic convenience functions around scipy signal filters

mhealth.generic.filters.butterworth(arr: numpy.ndarray, cutoff: Union[float, Tuple[float, float]], freq: float, order: int = 5, ftype: str = 'highpass') → numpy.ndarray¶

Butterworth filters the array through a highpass butterworth filter.

Parameters

  • arr (list or numpy.array) – The timeseries array on which the filter is applied

  • cutoff (scalar or len-2 sequence) – The critical frequencies for the Butterworth filter. The point at which the gain drops to 1/sqrt(2) of the passband. Must be length-2 sequence for a bandpass filter giving [low, high]. Or else the scalar cutoff frequency for either a low-pass’ or ‘highpass’ filter.

  • freq (float) – The frequency (Hz) of the input array

  • order (int) – The order of the filter.

  • ftype (str - {'highpass', 'lowpass', 'bandpass'}) – The filter type. Default is ‘highpass’

Returns

Filtered input signal

Return type

np.array
