Calculate location distribution-based features Typically each feature has two functions - one that takes a dataframe and the other that takes numpy ndarrays. The dataframe is assumed to have a datetime index and a latitude and longitude column, each measured in degrees. Cluster labels should be integers from 0 upwards. Non-clustered points should be labelled -1.
